Global distance computation

In order to perform global comparison of structures take the following steps (in the given order):
  1. Open structure(s) from files (PDB or mmCif)
  2. Select structures to compare
  3. Select distance measure
  4. Compute distance matrix
  5. Visualise or cluster results

Local distance(s) computation

  1. Open structure(s) from files (PDB or mmCif)
  2. Select chain(s) to compare
  3. Configure the comparison properties
  4. Calculate differences of torsion angles for specified chains or for whole structures

Sequence alignment

  1. Open structure(s) from files (PDB or mmCif)
  2. Select chain(s) to align
  3. Select global (Smith-Waterman) or local (Needleman-Wunsch) alignment method
  4. Compute alignment

Sequence alignment is performed with the following parameters:

  • Open penalty: 1
  • Extension penalty: 10
  • Substitution matrix for nucleic acids: NUC 4.4
  • Substitution matrix for proteins: BLOSUM62

Structure alignment

  1. Open structure(s) from files (PDB or mmCif)
  2. Select chain(s) to compare
  3. Compute alignment
