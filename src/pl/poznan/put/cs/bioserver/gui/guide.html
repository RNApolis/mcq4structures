MCQ4Structures allows to:
  • compute global or local distances between structures of RNAs/proteins
  • align sequences or 3D structures of RNAs/proteins.
Here you can find basic instructions for using our tool in the above processes.

Global distance computation

Global comparison can be performed either in torsional angle space (MCQ value determines the distance) or in Cartesian space (RMSD value shows the distance). In order to compare structures globally, take the following steps:
  1. Open structure(s) from file(s) (PDB or mmCif format): File→Open structure(s)
  2. Select structures to compare: Distance measure→Select structures to compare
  3. Select distance type: Distance measure→Global MCQ or Distance measure→Global RMSD
  4. Compute distance matrix: Distance measure→Compute distance(s)
  5. Visualise or cluster results: Distance measure→Visualise results or Distance measure→Cluster results
In order to save the results of global distance computation use File→Save results. The results are saved to csv file.

Local distance(s) computation

Local comparison is performed in torsional angle space. In order to compute a vector of local distances, i.e. distances between the corresponding angles identified along the structure, take the following steps:
  1. Open structure(s) from file(s) (PDB or mmCif format): File→Open structure(s)
  2. Select structures and chain(s) to compare: Distance measure→Select structures to compare
  3. Select torsion angle(s) for distance computation: Distance measure→Select torsion angles
  4. Compute distance vector: Distance measure→Compute distance(s)
  5. Visualise results: Distance measure→Visualise results
In order to save the results of local distance computation use File→Save results. The results are saved to csv file.

Sequence alignment

In order to perform sequence alignment (global or local) take the following steps:
  1. Open structure(s) from file(s) (PDB or mmCif format): File→Open structure(s)
  2. Select structures and chain(s) to align: Alignment→Select structures to align
  3. Select alignment method:
    • Global sequence alignment (Smith-Waterman): Alignment→Global sequence alignment
    • Local sequence alignment (Needleman-Wunsch): Alignment→Local sequence alignment
  4. Compute alignment: Alignment→Compute alignment
Sequence alignment is performed with the following parameters:
  • Open penalty: 1
  • Extension penalty: 10
  • Substitution matrix for nucleic acids: NUC 4.4
  • Substitution matrix for proteins: BLOSUM62
In order to save the results of sequence alignment use File→Save results. The results are saved to TXT file.

3D Structure alignment

In order to perform 3D structure alignment take the following steps:
  1. Open structure(s) from file(s) (PDB or mmCif format): File→Open structure(s)
  2. Select structures and chain(s) to align: Alignment→Select structures to align
  3. Select 3D structure alignment: Alignment→3D structure alignment
  4. Compute alignment: Alignment→Compute alignment
In order to save the results of 3D structure alignment use File→Save results. The results are saved to PDB file.

Remarks

  1. In case of processing multi-model pdb file only the first model is opened by MCQ4Structures.
  2. If the open structure contains residues/atoms with more than one alternative position (coordinate set) only the first one is considered.
