Global distance computation

Global comparison can be performed either in torsional angle space (MCQ value determines the distance) or in Cartesian space (RMSD value shows the distance). In order to compare structures globally, take the following steps (in the given order):
  1. Open structure(s) from file(s) (PDB or mmCif format)
  2. Select structures to compare
  3. Select distance measure: MCQ (default) or RMSD
  4. Compute distance matrix
  5. Visualise or cluster results

Local distance(s) computation

Local comparison is performed in torsional angle space. In order to compute a vector of local distances, i.e. distances between the corresponding angles identified along the structure, take the following steps (in the given order):
  1. Open structure(s) from file(s) (PDB or mmCif format)
  2. Select chain(s) to compare
  3. Select torsion angle(s) for distance computation
  4. Compute distance vector

Sequence alignment

In order to perform sequence alignment take the following steps:
  1. Open structure(s) from file(s) (PDB or mmCif format)
  2. Select chain(s) to align
  3. Select alignment method: global (Smith-Waterman) or local (Needleman-Wunsch)
  4. Compute alignment

Sequence alignment is performed with the following parameters:

  • Open penalty: 1
  • Extension penalty: 10
  • Substitution matrix for nucleic acids: NUC 4.4
  • Substitution matrix for proteins: BLOSUM62
